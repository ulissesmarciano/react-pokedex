# React Pokédex

Projeto de uma pokédex testando habilidades de front-end e habilidades de requisição de API's.

## Tecnologias usadas
- HTML
- CSS
- JS
- React Router Dom
- Styled Components
- React tabs

## Como Utilizar

1 - Clone o projeto

```
git clone <url>
```
2 Acesse a pasta do projeto

```
cd repositorio-exemplo
```
